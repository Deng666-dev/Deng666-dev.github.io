Posted on 2025-12-19 Edited on 2025-12-31 In 开发笔记 Word count in article: 1.8k Reading time ≈ 2 mins.

命令

hexo clean 删除本地生成的public文件
hexo g 生成html文件
hexo s 启动预览本地服务器
hexo d 推送到Github中

写入新文章
hexo new “文章标题”
启动预览
hexo cl && hexo s
部署上线
hexo cl && hexo g && hexo d
创建页面
hexo new page categories
