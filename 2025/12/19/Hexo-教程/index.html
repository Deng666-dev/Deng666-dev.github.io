Hexo 教程

Created2025-12-19|Updated2025-12-22|网站主题搭建
|Post Views:

命令

hexo clean 删除本地生成的public文件
hexo g 生成html文件
hexo s 启动预览本地服务器
hexo d 推送到Github中

写入新文章
hexo new “文章标题”
启动预览
hexo cl && hexo s
部署上线
hexo cl && hexo g && hexo d
创建页面
hexo new page categories

目录

\Blog\my-blog\ 目录:
_config.yml 站点配置
-config.butterfly.yml 主题配置
/source/_posts 文章仓库
/source/img/ 图床
